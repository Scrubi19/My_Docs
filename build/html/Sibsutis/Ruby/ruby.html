Ruby on Rails¶

Установка на Ubuntu¶

  1. Установить rbenv

git clone https://github.com/rbenv/rbenv.git ~/.rbenv

  1. Вписать зависимости

echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(rbenv init -)"' >> ~/.bashrc
exec $SHELL

  1. Установить ruby-build

git clone https://github.com/rbenv/ruby-build.git "$(rbenv root)"/plugins/ruby-build

  1. Установить ruby

rbenv install 2.7.1
rbenv global 2.7.1

Пример Gemfile:

source 'https://rubygems.org'

ruby '2.7.1'

gem 'colorize'
gem 'csv'
gem 'rails'
gem 'rspec'
gem 'rubocop'
gem 'simplecov'

Команды:

bundle install - установка gem-ов из Gemfile
bundle exec rubocop - запуск rubocop
bundle exec rspec - запуск тестов

Начало¶

Ruby on Rails – это фреймворк для разработки веб-приложений, написанный на языке программирования Ruby

rails new <name of project> # генерирует новый проект на rails
rails server # создает локальный сервер для страницы проэкта (http://localhost:3000/)
